# TypeScript Shopping Cart (ts-cart)
[![Check Code](https://github.com/Milka79rus/ts-cart/actions/workflows/run.yml/badge.svg)](https://github.com/Milka79rus/ts-cart/actions/workflows/run.yml)


Проект представляет собой реализацию логики корзины интернет-магазина на TypeScript. Проект включает в себя работу с интерфейсами, классами, а также модульное тестирование с использованием Jest.

## Реализованные возможности

### Задание 1: Movie
Реализован класс `Movie`, описывающий фильм. Класс имплементирует интерфейс `Buyable` и содержит следующие поля:
* `id` (уникальный идентификатор)
* `name` (название)
* `price` (цена)
* `year` (год выпуска)
* `country` (страна)
* `slogan` (слоган)
* `genre` (жанр)
* `time` (продолжительность в минутах)

### Задание 2: Методы корзины
Класс `Cart` расширен дополнительными методами для управления покупками:
* `getTotalPrice()` — расчет суммарной стоимости всех товаров в корзине без учета скидок.
* `getTotalPriceWithDiscount(discount)` — расчет суммарной стоимости с учетом переданной скидки (в процентах).
* `removeById(id)` — удаление товара из корзины по его идентификатору.

### Задание 3: Особенности логики (со звездочкой)
Реализована дифференцированная логика добавления товаров в корзину:
* **Медиа-контент (Фильмы, Книги)**: При попытке добавить один и тот же фильм несколько раз, в корзине сохраняется только одна единица товара.
* **Физические товары (Смартфоны)**: Реализована возможность добавления нескольких единиц одного и того же товара. Для этого используется свойство `canCount`.
* Добавлен метод `decrease(id)`, позволяющий уменьшать количество конкретного товара в корзине.
